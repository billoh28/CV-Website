Sign Language Translator - Third Year Project

As our third year project, my project partner and I wrote and trained Neural Networks to translate Irish Sign Language from prerecorded video into audio.

The program we produced utilised Convoluted Neural Networks (CNN) and computer vision libraries, written in Python 3.


The functions of this program are as follows:

  • Converts a video into frames, applying image processing to convert the frames into multi-dimensional arrays and feed them to a trained CNN model.

  • Accumulates the output of the CNN models to plain text.

  • Convert the text produced by the CNN into audio, and save the audio file into the current working directory.


A graph and confusion matrix depicting this result is show on the right of the page.

This communication channel has the potential to greatly decrease the impacts of deafness, and other communication disorders, which could reduce loneliness and isolation in these communities.

We received a first class honours in this project.

Network API - Alternate Intra

This was part of a DCU group project meant to replicate work placement for those who lost their jobs due to COVID-19.

I worked alongside a fellow software engineering student to develop a Network API for a network of smart hand sanitiser devices. The API was wrote in Python 3 using Flask as a framework.

Functions of the API:

  • Records usage statistics of the individual hand sanitiser dispensers.

  • Performs data analytics on the usage statistics.

  • Displays status of diapensers, usage statistics and data analytics report at specified endpoints requiring an API key.

The GitLab repository this code is contained within is private as it contains Firebase credentials
