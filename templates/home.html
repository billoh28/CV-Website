Projects


Orthodox MMA Trainer - Final Year Thesis


For my final year thesis, my project partner and I created a progressive web application which utilised pose estimation and computer vision to predict a user's technique / form in regards to certain strikes used within mixed martial arts, such as a jab.

We created datasets by recording poor and proper technique, reducing the dimensioanlity of the data and then feeding this data to BlazePose. This is a pose estimation pipeline which was cutting edge at the time. It produces pose estimations, frame-by-frame, in the form of 32 pose landmarks. This data was then used to create datasets and train multiple classifiers.

The web application was written using Flask, Python, Javascript (used for video encoding), HTML and CSS, and was hosted on an AWS EC2 instance. GoLang was also used to produce an API for storing feedback reports produced by users and user authentification.

For more detail, along with a user guide and technical specification please refer to the Project Github Repository

The web application is available through the following link: Orthodox MMA



Sign Language Translator - Third Year Project


As our third year project, my project partner and I designed and trained Neural Networks to translate Irish Sign Language from prerecorded video into audio.

The program we produced utilised Convoluted Neural Networks (CNN) and computer vision libraries, such as Keras and Numpy, and was written in Python.


The functions of this program are as follows:

  • Converts a video into frames, applying image processing to convert the frames into multi-dimensional arrays and feed them to a trained CNN model.

  • Accumulates the output of the CNN models to plain text.

  • Convert the text produced by the CNN into audio, and save the audio file into the current working directory.


The graph and confusion matrix below depict the accuracy of the ensemble model. The orange line in the graph shows the number of correct predictions for that letter sign pair in a test dataset, while the blue line is the number of incorrect predictions for that pair.

This communication channel has the potential to greatly decrease the impacts of deafness, and other communication disorders, which could reduce loneliness and isolation in these communities.

We received a first class honours in this project.
